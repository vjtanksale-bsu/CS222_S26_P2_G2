# CS222 Project 2 – Course Scheduling System
# Project Overview
This program allows students to register for courses by entering course numbers. It loads available courses from data files, validates user input, and outputs a formatted schedule with assigned sections.

# Features
Load course data from multiple files
Validate and register courses by course number
Automatically assign a section for each selected course
Display a clean, formatted course schedule
